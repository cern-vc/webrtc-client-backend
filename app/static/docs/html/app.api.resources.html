Class that manages the browser handling API

mediatypes(resource_cls)¶

post()¶

Tries to make a request to the CERN Oauth2 API Url and retrieve some user information with the given user token. If the token is not valid means that the user is not valid and will be rejected.

Returns:The generated token for the user or an error.
